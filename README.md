# 💡 Smart Streetlight Energy Prediction

A Machine Learning project developed to predict streetlight energy consumption and support smart city energy optimization. The system uses historical data and predictive analytics to estimate future energy usage for efficient resource management.

---

## 🚀 Features

- Predict streetlight energy consumption using Machine Learning
- Analyze historical usage patterns
- Support smart city energy optimization
- Improve energy efficiency planning
- Data preprocessing and model training
- Performance evaluation and visualization

---

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook / Python Script

---

## 📂 Project Files

main.py / notebook.ipynb - Main project file  
dataset.csv - Input dataset  
requirements.txt - Dependencies  
README.md - Project Documentation

---

## ▶️ Run Locally

1. Clone the repository

git clone https://github.com/vinay-0208/smart-streetlight-energy-prediction.git

2. Install dependencies

pip install -r requirements.txt

3. Run the project

python main.py
